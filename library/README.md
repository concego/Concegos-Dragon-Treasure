# Biblioteca de Jogos CDT (Concego's Dragon Treasure)

Bem-vindo à biblioteca central de jogos J-SOM para o ecossistema CDT. Aqui você encontra títulos verificados e apoiados pela equipe **Eu Concego Jogar (ECJ)**.

## 1. Como usar
Para jogar, basta baixar o arquivo `.json` desejado e inseri-lo no console via botão **"Inserir Jogo"** no seu PWA do CDT.

## 2. Estrutura da Biblioteca
*   **/official:** Jogos desenvolvidos ou rigorosamente testados pela equipe ECJ. Garantia total de acessibilidade e diversão inclusiva.
*   **/community:** Títulos enviados pela comunidade que seguem as diretrizes do SDK e foram aprovados para compor o catálogo inicial.

## 3. Quer enviar seu jogo?
Se você desenvolveu um jogo J-SOM e quer o selo de apoio da ECJ:
1. Verifique se o jogo segue todas as [Diretrizes de Inclusão](DEVELOPER_GUIDE.md).
2. Entre em contato com a equipe ou envie um Pull Request para este repositório.
3. Após o teste de qualidade, seu jogo será adicionado à pasta `/community` ou `/official`.

## 4. Desenvolvimento Independente
O CDT é uma plataforma aberta. Você é livre para criar e compartilhar seus jogos diretamente com amigos e comunidades para testes e feedback sem a necessidade de passar pela biblioteca oficial. Encorajamos a experimentação!
