# Concego's Dragon Treasure (CDT) 🐉💎
**O Console Virtual 32-bit para Inclusão Digital Nativa.**

O **CDT** é uma plataforma de jogos (PWA) criada pela equipe **Eu Concego Jogar (ECJ)**. Ele une a nostalgia estética dos anos 90 com um motor de acessibilidade revolucionário que coloca cegos e videntes em pé de igualdade através do toque, som e vibração.

## 🚀 Como Jogar
Acesse o console diretamente pelo link do GitHub Pages (em breve) e instale o PWA no seu dispositivo Android ou iOS.

## 📁 Estrutura do Repositório
- **/console**: Código-fonte do motor (Kernel) e interface do console.
- **/sdk**: Kit de Desenvolvimento para criação de jogos no formato J-SOM.
- **/library**: Catálogo de jogos oficiais e da comunidade.
